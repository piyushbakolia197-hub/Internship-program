# 🌟 Internship Application Website

A simple, responsive, and user-friendly **Internship Application Website** built using **HTML**, **CSS**, and **JavaScript**. The website provides details about an internship program and allows candidates to submit their applications through an interactive form.

---

## 📖 Project Overview

This project is designed to demonstrate front-end web development skills by creating a responsive internship application portal. It includes an informative landing page and a validated application form for collecting applicant details.

---

## ✨ Features

* 📱 Responsive design for desktop and mobile devices
* 🎓 Internship information section
* 📝 Application form for candidates
* ✅ Client-side form validation using JavaScript
* 🎨 Clean and modern user interface
* ⚡ Lightweight with no external dependencies

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (ES6)**
